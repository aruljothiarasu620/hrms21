# HRMS 21 - Employee Portal

A premium, responsive HR and Payroll management web application.

## Features
- **Employee Self-Service (ESS)**: Profile management, salary slips, and attendance.
- **Attendance Dashboard**: Real-time tracking of presence and leave.
- **Recruitment & Training**: Manage hiring pipelines and skill development.
- **Modern UI**: Clean, high-contrast interface designed for productivity.

## Getting Started
Simply open `hr_portal_website.html` in any modern web browser.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Google Fonts (Plus Jakarta Sans)
- Font Awesome Icons
